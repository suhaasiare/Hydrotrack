Hydro Tracker is a **web-based water intake tracker** designed to help users monitor and manage their daily hydration. The application allows users to log water consumption, visualize progress toward daily goals, and maintain a history of intake â€” all in a **single, lightweight HTML file** with embedded CSS and JavaScript.  

---

## **Features**

- Log daily water intake in milliliters (ml).  
- Real-time **progress bar** showing percentage of daily goal achieved.  
- **Daily intake log** with timestamps.  
- **Reset functionality** to clear daily intake and logs.  
- **Responsive design**, works on both desktop and mobile browsers.  
- Lightweight and self-contained â€” no installation or external dependencies required.  

---

## **Demo**

You can open `index.html` in any modern web browser to try Hydro Tracker instantly.  

---

## **Functional Requirements**

- Users can input water intake.  
- The system updates the total daily consumption in real-time.  
- Progress toward daily goal is visually displayed.  
- Logs of all entries are maintained with timestamps.  
- Users can reset daily intake and logs as needed.  

---

## **Non-Functional Requirements**

- Fully responsive and user-friendly interface.  
- Efficient with minimal CPU and memory usage.  
- Runs entirely in a web browser without backend or server.  
- Compatible with modern browsers (Chrome, Firefox, Edge, Safari).  

---

## **Technical Details**

- Implemented in **HTML, CSS, and JavaScript** in a single file.  
- Optional use of **localStorage** for persisting daily water intake across sessions.  
- Real-time updates for progress bar and logs.  

---

## **Usage Instructions**

1. Clone or download this repository.  
2. Open `index.html` in a web browser.  
3. Enter the amount of water consumed in the input field.  
4. Click **Add Water** to log intake.  
5. Track your daily progress via the **progress bar**.  
6. Click **Reset** to clear all entries for the day.
